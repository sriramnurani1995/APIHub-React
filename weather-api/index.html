Welcome to the Weather Forecast API

This API provides weather data either for a given date or for an entire month. You can request weather data by providing the date in the format YYYY-MM-DD, or a month in the format YYYY-MM.

How to Use the Weather API

Use the following parameters to query the weather data:

Parameter Expected Value Default Behavior
date YYYY-MM-DD (e.g., 2024-12-08) Current Date (if no date is provided)
month YYYY-MM (e.g., 2024-12) Current Month (if no month is provided)

If you don't provide a date or month, the API will return the weather for the current date or month.

Example Request

            GET /api/weather?date=2024-12-08
        
            GET /api/weather?month=2024-12

These will return the weather conditions for the given date or month.
