Paragraph Placeholder API

Generate dynamic placeholder paragraphs for your projects.

API Usage

Use the following endpoint to generate paragraphs:

GET /api/paragraphs?type=&length=&count=

Example

/api/paragraphs?type=cat&length=medium&count=3

Parameters

Below is a detailed explanation of the API parameters:

Parameter Description Accepted Values Default
type The type of placeholder text to generate. lorem, cat, pup, business, tech, hipster lorem
length The length of the paragraphs. short, medium, long medium
count The number of paragraphs to generate. 1 to 10 1
