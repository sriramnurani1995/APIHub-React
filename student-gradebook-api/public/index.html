Student Gradebook API

A scalable and lightweight API for managing and retrieving student grades.

Introduction

This API provides two primary endpoints to retrieve information about course gradebooks: header information and a complete gradebook breakdown for each course.

API Endpoints

Endpoint Description Example
GET /api/header/:courseId Retrieves the header information, including the components and type weightage for a course. GET /api/header/CS500
GET /api/gradebook/:courseId Retrieves the complete gradebook for a course, including individual student performance, weighted percentages, and final grades. GET /api/gradebook/CS500
